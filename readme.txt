OpenCabs taxi bookings snippet creator.

== Description ==
OpenCabs taxi bookings snippet creator. It allows registered clients to create their own booking/quote/singup snippets and place them anywhere on the page.

== Installation ==
1. Upload plugin folder to the \"/wp-content/plugins/\" directory.
2. Activate the plugin through the \"Plugins\" menu in WordPress.
3. Follow login/setup process at Opencabs menu in admin panel.
